# Timbrature - progetto completo

Questa è l'app **Timbrature**, per gestire timbrature, promemoria e straordinari.

## Funzionalità principali
- Promemoria automatici: Lun–Ven
  - Entrata: 7:50
  - Uscita: 12:00
  - Entrata: 13:50
  - Uscita: 18:50
- Visualizzazione ultime timbrature
- Correzione manuale timbrature
- Widget home personalizzato
- Esportazione CSV mensile
- Calcolo straordinari oltre le 8 ore giornaliere

## Istruzioni per ottenere APK debug
1. Carica tutti i file del progetto in questo repository su GitHub (puoi usare l’app mobile o il browser).
2. Vai nella tab **Actions** → vedrai il workflow `Android CI - build debug`.
3. Premi **Run workflow** → si compilerà l’APK.
4. Al termine, sotto **Artifacts**, scarica `app-debug.apk`
